# TeamTime

TeamTime is a web-based project management platform developed as a Software Engineering project.

## 🚀 Project Goal

The aim of TeamTime is to help teams manage their projects from a single platform by providing:

- Project creation
- Team management
- Task management
- Progress tracking
- File sharing

## 🛠 Technologies

### Frontend
- React
- TypeScript
- Vite

### Backend (Planned)
- Java
- Spring Boot

### Database
- PostgreSQL
